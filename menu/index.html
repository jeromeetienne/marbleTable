What about Marble Table

It has been written by @jerome_etienne. It uses three.js with tQuery API for 3D and cannon.js for physics

You are controlling a Marble on a table!

Goal

The goal is to reach the checkerboard without touching the grey spheres as faster as you can. Each time you touch the grey spheres, the game start again.

How to play

Use Arrows to move the ball. Use 1 to 4 to move the camera position.
